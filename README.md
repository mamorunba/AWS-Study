# AWS-Study
## AWSが提供するセキュリティサービス
- 認証、認可、アクセス管理、ポリシー管理のサービス
- AWS操作のためのグループ・ユーザー・ロールの作成が可能
- グループ、ユーザーごとに、実⾏出来る操作を規定できる（IAMポリシー）
